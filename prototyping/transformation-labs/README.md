## Equipment

What sort of equipment is needed for a Transformation Lab?

### Basic

 - Workbench
 - Stools
 - Battery powered Drill / Screwdriver
 - Drill bits
 - Rotary tool
 - Pliers
 - Power supply
 - Magnifier
 - Articulated desk lamp
 - Multimeter
 - Precision screwdriver set
 - Storage / Organisers / Drawers
 - Cutting mat
 - Box cutter

### Intermediary

 - Screens (HDMI, VGA, composite video)
 - Speaker
 - Cable tester

### Advanced

 - e-valudata ;)
 - 3d printer
 - 3d scanner
 - lasercutter
